# 프로젝트 구현 범위

## 1. 수업에서 구현할 범위

본 수업에서는 실제 장비 연동이 아니라, 수업용 시뮬레이터와 Django 기반 대시보드를 중심으로 전체 데이터 흐름을 구현한다.

### 화면 계층

- Django Template 기반 대시보드
- 센서 상태 카드
- 가스 농도 차트
- 전력 사용량 차트
- 작업자 위치 표시 영역
- 알람 리스트
- AI 예측 결과 표시 영역
- RAG 대응 매뉴얼 표시 영역

### 데이터 계층

- 센서 데이터 계약 정의
- Raw/Staging/Mart 테이블 설계
- 유해가스 데이터 저장
- 전력 데이터 저장
- 작업자 위치 데이터 저장
- 알람 이력 저장
- 피드백 이력 저장
- AI 결과 저장

### 처리 계층

- Sensor Simulator
- Collector API
- Raw 저장
- Staging 검증
- Mart 생성
- 위험도 판단
- 알람 생성
- WebSocket 실시간 표시

### AI/RAG 계층

- AI 이상탐지 결과 저장
- 알람 유형별 대응 매뉴얼 검색
- RAG 검색 결과 대시보드 표시

### 품질/운영 계층

- 중복 데이터 확인
- 지연 데이터 확인
- 오류 데이터 보관
- 데이터 품질 리포트
- 최종 시연 문서화

## 2. 수업에서 축소할 범위

다음 기능은 수업용으로 축소하거나 시연형으로 구현한다.

- 실제 센서 장비 연동
- 실제 폐쇄망 인프라
- 정밀 실내 측위 알고리즘 전체 구현
- Kafka/Spark 대규모 운영
- Kubernetes 상용 배포
- AI 재학습 자동화
- ISO45001 전체 업무 프로세스
