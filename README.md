# 디코나이 AI 네이티브 산업안전 관제 플랫폼

## 현재 단계

현재 단계는 `0단계. 프로젝트 기준선 정리`이다.

현재 프로젝트는 Django 기반으로 초기화되어 있으며, HTML/CSS/JavaScript 기반 대시보드 화면이 Template과 Static 구조로 연결되어 있다.

아직 데이터 모델, 수집 API, WebSocket, AI 추론, RAG 기능은 구현되어 있지 않다.

## 0단계 목표

0단계에서는 실제 기능 구현 전에 다음 기준을 확정한다.

- 프로젝트 목적
- 수업용 구현 범위
- 센서 데이터 구조
- 가스별 임계치
- 샘플 이벤트
- 최종 시연 시나리오
- 데이터 흐름도
- 패턴 적용 맵
- 브랜치 개발 로드맵

## 실행 방법

```bash
python manage.py runserver
```

브라우저에서 대시보드 화면이 정상적으로 표시되는지 확인한다.
